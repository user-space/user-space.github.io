PFLogLevel

enum PFLogLevel : uint8_t {}

PFLogLevel enum specifies different levels of logging that could be used to limit or display more messages in logs.

See

Parse.+setLogLevel:

See

Parse.+logLevel
  • PFLogLevelNone

    Log level that disables all logging.

    Declaration

    Objective-C

    PFLogLevelNone = 0
  • PFLogLevelError

    Log level that if set is going to output error messages to the log.

    Declaration

    Objective-C

    PFLogLevelError = 1
  • PFLogLevelWarning

    Log level that if set is going to output the following messages to log: - Errors - Warnings

    Declaration

    Objective-C

    PFLogLevelWarning = 2
  • PFLogLevelInfo

    Log level that if set is going to output the following messages to log: - Errors - Warnings - Informational messages

    Declaration

    Objective-C

    PFLogLevelInfo = 3
  • PFLogLevelDebug

    Log level that if set is going to output the following messages to log: - Errors - Warnings - Informational messages - Debug messages

    Declaration

    Objective-C

    PFLogLevelDebug = 4
